Stay safe from spam, scams and phishing.

SpamGuard is a browser-based spam detection tool that analyzes any message and instantly tells you how risky it looks.

Why SpamGuard?

• 100% frontend – built using HTML, CSS & modern JavaScript (ES6+).
• Explains exactly why a message is flagged as spam.
• Perfect for students, demonstrations and portfolios.

Key Features

Simple interface, powerful detection logic.

Rule-based Scoring

Checks suspicious keywords, urgency and links to generate a transparent spam score.

History Tracking

Your last few analyses are stored in the browser using localStorage.

Responsive Layout

Works smoothly on laptop and mobile screens with a clean minimal design.
