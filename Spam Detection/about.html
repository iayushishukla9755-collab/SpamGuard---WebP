About SpamGuard

SpamGuard is a JavaScript-based interactive web application that demonstrates spam detection using modern frontend features.

Project Aim

To build a fully client-side spam detection website that uses DOM manipulation, events, async programming and browser storage.

What it Detects

• Suspicious words (free, win, lottery, offer, urgent)
• Scam-style urgency.
• Malicious-looking links.

How it is Built

Implemented using HTML, CSS and ES6+ JavaScript with array methods (map, filter, reduce), async/await, Fetch API and localStorage.
